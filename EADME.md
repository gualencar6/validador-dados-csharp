# 🧠 Validador Inteligente de Dados

Projeto desenvolvido em C# com o objetivo de validar diferentes tipos de dados, como:

* CPF
* Endereço IP
* Senhas

## 🚀 Tecnologias utilizadas

* C#
* .NET
* Console Application (ou API, se você já fez)

## 📌 Funcionalidades

* Validação de CPF com regras oficiais
* Validação de IP (formato correto)
* Validação de senha (critérios de segurança)

## 💡 Objetivo do projeto

Este projeto foi criado com foco em praticar lógica de programação, validações e estruturação de código em C#.
